= Troubleshooting

To troubleshoot migration errors, you can check the xref:Migration_logs[migration logs] and the following possible causes:

* xref:Infrastructure_mapping_errors[Infrastructure mapping errors]
* xref:Migration_plan_errors[Migration plan errors]
* xref:IP_address_errors[Postmigration virtual machine IP address errors]
* Environment configuration errors:
** xref:VMware_environment_errors[VMware]
** xref:RHV_VM_migration_failure[Red Hat Virtualization]
** xref:OSP_VM_migration_failure[OpenStack Platform]

xref:Known_issues[] provides information about critical updates and issues that will be addressed in a future release.
