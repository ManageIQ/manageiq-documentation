
[[start-a-vm]]
=== Start a VM

==== Request:

----
POST /api/vms/1921
----

[source,json]
----
{
  "action": "start"
}
----

==== Response:

[source,json]
----
{
  "success": true,
  "message": "VM id:1921 name:'aab_demo_vm' starting",
  "task_id": 610,
  "task_href": "http://localhost:3000/api/tasks/610",
  "href": "http://localhost:3000/api/vms/1921"
}
----

=== Querying task progress:

==== Request:

----
GET /api/tasks/620
----

==== Response:

----
{
  "href": "http://localhost:3000/api/tasks/610",
  "id": 610,
