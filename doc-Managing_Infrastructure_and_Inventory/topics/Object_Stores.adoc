[[object-stores]]
= Object Stores

Navigate to menu:Compute[Clouds > Object Stores] to see a list of all cloud object stores along with information including _Key_, _Size (bytes)_, _Object Count_, _Cloud Tenant_, and _Cloud Provider_. Click on an object store to see its properties and relationships with the cloud provider, tenant, and object store on the summary page.
