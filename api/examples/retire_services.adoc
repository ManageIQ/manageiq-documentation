
[[retire-services]]
=== Retire Multiple Services

==== Request:

----
POST /api/services
----

[source,json]
----
{
  "action" : "retire",
  "resources" : [
      { "href" : "http://localhost:3000/api/services/100" },
      { "href" : "http://localhost:3000/api/services/101", "date" : "11/01/2015", "warn" : "4" },
      { "href" : "http://localhost:3000/api/services/102", "date" : "11/02/2015", "warn" : "4" }
  ]
}
----
