= Creating a Virtual Machine Comparison Report

Output the data from a comparison report in TXT, CSV or PDF formats.

. Create the comparison for the report.
. Click the output button for the chosen report type.
+
* Click  image:2133.png[] (*Download comparison report in TXT format*) for a text file.
* Click  image:2133.png[] (*Download comparison report in CSV format*) for a CSV file.
* Click  image:2134.png[] (*Download comparison report in PDF format*) for a PDF file.
