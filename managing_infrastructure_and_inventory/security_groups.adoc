[[security_groups]]
= Security Groups

You can group instances using security groups to restrict port or IP address accessibility.
Security groups are to be created from the cloud provider side and can be assigned to instances using {product-title} instance provisioning. 

Cloud providers that currently support this function include: Amazon EC2, OpenStack, and Enterprise Virtualization.
